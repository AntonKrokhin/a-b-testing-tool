# A/B Testing Tool

## Project Objective
The objective of this project is to automate the calculation of statistical significance for A/B testing experiments. By engineering a Python script, the tool automatically evaluates conversion lifts across variant groups, calculates statistical thresholds, and outputs the final metrics into an interactive Tableau dashboard for business stakeholders.
